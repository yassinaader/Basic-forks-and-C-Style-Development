# Assignment 2 — Basic Forks & C-Style Development

This repository contains the required programs, Makefile, explanations, and licensing for Assignment 2.

## Contents
- `pro_creation.c` — Demonstrates fork() usage
- `file1.c` and `file2.c` — Demonstrate the role of the linker
- `simple_pro.c` — Used to inspect dynamic libraries via ldd
- `Makefile` — Builds all programs
- `answers.txt` — Contains explanations
- `LICENSE` — MIT License

## How to Compile
```
make
```

## How to Run
```
./pro_creation
./file_program
./simple_pro
```
